Company Docker Setup Script

This script automates the setup of Docker containers for your workstation:

  • Select the development environment you need (e.g., Java, IDE, database).
  • The script pulls the corresponding official Docker images from Docker Hub.
  • Automatically configures containers with ports, networking, and environment variables.
  • Optionally works offline with pre-included internal images if no internet is available.
  • Preconfigured and approved by the IT team for quick and consistent setup.
